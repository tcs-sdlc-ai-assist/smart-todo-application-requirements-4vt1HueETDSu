# Changelog

## [1.0.0] - 2024-06-01

### Added

- Initial release of **Smart Todo App** built with Vite and React.
- Core features:
  - Add, edit, delete, and complete todos.
  - Task details: title, description, due date, priority, recurrence, tags.
  - Dashboard view: Today, Upcoming, Completed tasks.
  - Task search, filter by status and priority.
  - LocalStorage persistence with versioning.
  - Accessible, responsive UI.
  - Modal dialog for product roadmap.
  - Placeholder UIs for upcoming features:
    - AI Suggestions
    - Collaboration
    - Integrations
    - Reminders & Recurring Tasks
  - In-browser notification support for reminders (permission required).
  - Unit tests for all major components and services.

### Setup

- Uses Vite for fast development and build.
- ESLint and Vitest for code quality and testing.
- No external state management or UI libraries—pure React and CSS modules.

For feedback or feature requests, see the Product Roadmap dialog in the app.
